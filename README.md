# 📊 Power BI Sales Dashboard

## Overview
This project is an interactive Sales Dashboard created using Microsoft Power BI. It helps visualize sales performance through KPIs and charts, making it easier to identify trends and business insights.

## Dashboard Features
- Total Sales
- Total Orders
- Total Customers
- Total Products
- Sales by Month
- Sales by Region
- Sales by Category
- Sales by Sub-Category
- Sales by Segment
- Sales by Ship Mode
- Interactive Slicers (Year, Region, Month)

## Tools Used
- Microsoft Power BI
- Microsoft Excel/CSV
- Data Visualization

## Dataset
The dashboard is built using a sales dataset containing information about orders, customers, products, regions, categories, ship mode, and sales.
